# Minecraft - LambDynamicLights/TerraFirmCraft Compatibility Resourcepack #

A datapack which provides compatibility between the LambDynamicLights family of
mods and TerraFirmCraft.

## About ##

This mod provides compatibility between
[TerraFirmaCraft](https://www.curseforge.com/minecraft/mc-mods/terrafirmacraft)
and the [LambDynamicLights](https://www.curseforge.com/minecraft/mc-mods/lambdynamiclights)
family of dynamic lighting mods.

Such mods include:

* [LambDynamicLights](https://www.curseforge.com/minecraft/mc-mods/lambdynamiclights)
  (kind of obvious).
* [Sodium/Embeddium Dynamic Lights](https://www.curseforge.com/minecraft/mc-mods/dynamiclights-reforged)
  (a Sodium/Embeddium-compatible port of LambDynamicLights).
* [LambDynamicLights (Unofficial NeoForge)](https://www.curseforge.com/minecraft/mc-mods/lambdynamiclights-unofficial-neoforge)
  (LambDynamicLights unofficially ported to NeoForge).
* [Celeritas Dynamic Lights](https://www.curseforge.com/minecraft/mc-mods/celeritas-dynamic-lights)
  (Sodium/Embeddium Dynamic Lights ported to Celeritas for 1.12.2).
* [Ryoamic Lights](https://www.curseforge.com/minecraft/mc-mods/ryoamiclights)
  (LambDynamicLights unofficial [Architectury](https://www.curseforge.com/minecraft/mc-mods/architectury-api) port).
* And many others.

Right now this resourcepack is guaranteed to support (as of May 9th, 2026) the
following combinations of mods:

* the latest version(s) of
  [TerraFirmCraft](https://www.curseforge.com/minecraft/mc-mods/terrafirmacraft).
* the latest version(s) of
  [Sodium/Embeddium Dynamic Lights](https://www.curseforge.com/minecraft/mc-mods/dynamiclights-reforged).

Other combinations of TerraFirmCraft + addons and LambDynamicLights mods should
work but your mileage may vary!

Feel free to report any bugs on the [GitHub issue tracker]()!

## Installation ##

1. Place in your Minecraft instance's `resourcepacks` folder.
2. Enable in-game.
    * Ensure it is above any other resourcepacks that alter dynamic lighting
      resources.
3. Profit!
